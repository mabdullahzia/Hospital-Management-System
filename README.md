# Hospital Management System

Console-based system developed in C++ to manage patient records, hospital departments, and statistics.

## Features
- Patient registration with details: name, age, department, disease, doctor, visit date
- Search patients by unique ID
- Department & disease statistics
- Menu-driven interface for smooth navigation

## Technologies Used
- C++ Classes & Objects
- Vectors for dynamic storage
- Modular programming and CLI

## How to Run
1. Open terminal
2. Compile: `g++ main.cpp -o HospitalSystem`
3. Run: `./HospitalSystem`
